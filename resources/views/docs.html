Panduan Mulai Cepat

Mulai gunakan SlugPost dalam 5 langkah sederhana:

  1. 1 Kunjungi halaman utama slugpost.com
  2. 2 Tempel konten markdown Anda atau unggah file .md
  3. 3 Pilih *slug* URL kustom (misalnya, "my-awesome-post")
  4. 4 Klik "Terbitkan Postingan"
  5. 5 Simpan tautan edit Anda! Tautan ini hanya ditampilkan sekali

Menerbitkan Konten

Dua cara untuk menerbitkan markdown Anda:

  • Tempel: Salin dan tempel markdown secara langsung
  • Unggah: Unggah .md, .markdown, atau .txt files

Format Slug: Hanya huruf kecil, angka, dan tanda hubung (misalnya, my-post-2025)

Mengedit Postingan

Gunakan tautan edit unik Anda untuk memperbarui konten:

⚠️ Penting

Tautan edit hanya ditampilkan sekali setelah penerbitan. Simpan segera di tempat yang aman!

Format: slugpost.com/slug-anda/edit/token

Kehilangan tautan edit Anda? Daftar dan klaim postingan untuk mendapatkan kembali akses.

Mengklaim Kepemilikan

Daftar untuk mengklaim postingan Anda dan dapatkan:

  • Dashboard untuk mengelola semua postingan
  • Edit tanpa memerlukan tautan edit
  • Atribusi penulis pada postingan
  • Melihat statistik dan analitik

Dukungan Markdown

Sintaks markdown penuh didukung:

  • • Judul (H1-H6)
  • • Tebal, miring, coret
  • • Daftar (berurutan & tidak berurutan)
  • • Tautan dan gambar
  • • Blok kode dengan penyorotan sintaksis
  • • Tabel
  • • Kutipan blok (*Blockquotes*)
  • • Garis horizontal
